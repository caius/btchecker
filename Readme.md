# BTChecker

Given a phone number, use a BT website API to look up if they have BT Infinity available, and if so whether it's enabled or not for said number.

## Usage

    $ btchecker <phone number>
    01952188188 served by GEOGRAPHICAL EXCHANGE (GGEX)
    Infinity service already enabled!
